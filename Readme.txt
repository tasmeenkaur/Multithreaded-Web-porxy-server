To run the application :
1. Type the following command in terminal: "python multithreadServer.py <port number>"
    If port number is not specified the default port number is 8080
2. Once it shows "ready to server" go to web browser and type "localhost:8080/www.github.com"
3. If present in cache, proxy server will read from cache. If not it will request server and also create a cached file and send it to client
4. log.txt file has all the logs includung length of bytes sent, length of bytes received, time elapsed, host address, host name, port number and detailed messages


Developmnt environment used:
1. Python 2.7.10
no additional packages required
